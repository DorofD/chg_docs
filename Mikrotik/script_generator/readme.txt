1) В PowerShell перейти в папку проекта, выполнить:
    python -m venv venv
    .\venv\Scripts\activate
    pip install -r requirements.txt 
    Виртуальное окружение готово, для выхода из него можно ввести deactivate

2) Сформировать Excel документ "file.xlsx" с необходимыми параметрами, пример в папке проекта

3) Изменить файл generator.py:
    Изменить названия столбцов во всех местах, где фигурирует лист. Например в sheet['ip_ap'][i], меняем 'ip_ap' на своё значение
    Вставить свой шаблон скрипта по аналогии с существующим в переменную script_template

4) Выполнить скрипт. В PowerShell можно "python generator.py". В папке проекта должны появиться файлы в соответствии с записями в Excel документе
